# Parasitoid sugar meta-analysis

This is the script accompanying the paper "Does additional sugars in the field improve biological control by parasitoids? A systematic review and meta analysis". It is now private but will be made public upon papers' acceptance.

## Data availability

All the data and associated metadata can be found in the relevant OSF project (**link will be pasted here upon the paper's acceptance**). Please note that you do not need to download the data to run the scripts; it will be downloaded automatically.

## Code execution

To reproduce analyses, you just need to open the R project `Parasitoid_sugar_meta_analysis.Rproj` in the main folder. To work on the script directly, you can then launch `R/main_script.R`. For a more detailed version, with additional comments and guidance at each step, you can launch `complete_analysis.Rmd` in the main folder. This will reproduce the Appendix S2 in the `.pdf` file accompanying the paper (**link will be placed here upon the paper's acceptance**).

## Session information

All analyses were performed using R version 4.4.1. Details about OS, R version and R packages can be found in `Outputs/session_info.txt`.

## Repository detailed description

- `Parasitoid_sugar_meta_analysis.Rproj` is the R project associated with the study. Just open this file in R to reproduce the whole analysis.
- `complete_analysis.Rmd` is an annotated RMarkDown file containing the whole analysis. It allow reproducing all outputs present in the paper and in the supplementary file associated with the paper. Running it will actually reproduce **Appendix S2** accompanying the paper in the SI, as a `.pdf` file.
- `references.bib` is a BibTex file containing references associated with the `.Rmd` document.
- **R** folder: contains all files needed to reproduce analyses.
  - `main_script.R` also generates the whole analysis needed to reproduce results produced in the paper and in supplementary files (less heavy, but with fewer annotations than the `.Rmd` file.
  - The **Functions** folder contains several `.R` custom functions needed to reproduce the analyses. Please refer to annotations in each script to see what they do.
- **Outputs** is the folder in which all output generated by the main script will be stored.
  - `session_info.txt` is already there contains all information about OS, R version and packages used.
  - All other files are figures (`.png`) or outputs from sensitivity analyses (`.rds` files) that may be generated by running the main script.
- By default, there is no **Data** folder: it will be created when running `complete_analysis.Rmd` or `main_script.R`. All needed data files will be automatically downloaded from the associated OSF project (**insert link later**).
- `.gitignore` contains information about files included in the version control.
